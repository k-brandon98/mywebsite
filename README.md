# MyWebsite

This project has been converted into a React application powered by Vite.

## Available scripts

- `npm install` to install dependencies
- `npm run dev` to start the local development server
- `npm run build` to build the production bundle
- `npm run preview` to preview the production build locally

## Notes

- The original static pages were backed up as `index.legacy.html` and `projects.legacy.html`.
- App sources live in `src/` and styles are in `src/styles/site.css`.
